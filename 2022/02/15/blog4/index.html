JDK、JRE、JVM

JDK : Java Development Kit

（java开发工具包）

JRE : Java Runtime Environment

（java运行时环境）

JVM : JAVA Virtual Machine

（java虚拟机）

卸载jdk步骤

1.删除java安装目录
2.删除环境变量中JAVA_HOME
3.删除path下java目录
4.java-version

下载jdk及配置

1.官网下载1.8安装包
2.配置环境变量 系统变量JAVA_HOME、目录
3.配置path变量
4.在cmd输入 java -version查看是否下载成功
