java数据类型

  • java为强类型语言，要求按严格要求使用变量，所有变量 都必须先定义，在使用。运行慢，但安全。
  • 弱类型语言则相反，没有严格的要求限制使用变量。运行快，但不太安全。

    八大基本数据类型

    整数
    • int a1 = 10;
    • byte a2 = 20;
    • short a3 = 30;
    • long a4 =40L;
      小数（浮点数）
    • float a5 = 1.25F;
    • double a6 = 2.354;
      浮点数只能表示有限的位数，位数大的二进制没有对应精确的值表示，会取最接近的数表示。
      所以最好要完全避免使用浮点数进行比较
      字符
    • char a7 = ‘a’ 单字符（字符串 String 属于类）
      布尔类型
    • boolean flag = true;

      常用进制

      • 二进制 int a1 = 0b10; 输出2
      • 八进制 int a2 = 010; 输出8
      • 十进制 int a3 = 10; 输出10
      • 十六进制 int a4 = 0x10; 输出16

        类型转换

        java是强类型语言，不同类型变量计算时要进行类型转换。
    • 强制转换
      （类型） 变量名 高–>低
    • 自动转换
      低–>高
      注意：不要溢出变量类型的取值范围
