java数据类型

  • java为强类型语言，要求按严格要求使用变量，所有变量 都必须先定义，在使用。运行慢，但安全。
  • 弱类型语言则相反，没有严格的要求限制使用变量。运行快，但不太安全。

    八大基本数据类型

    整数
    • int a1 = 10;
    • byte a2 = 20;
    • short a3 = 30;
    • long a4 =40L;
      小数（浮点数）
    • float a5 = 1.25F;
    • double a6 = 2.354;
      浮点数只能表示有限的位数，位数大的二进制没有对应精确的值表示，会取最接近的数表示。
      所以最好要完全避免使用浮点数进行比较
      字符
    • char a7 = ‘a’ 单字符（字符串 String 属于类）
      布尔类型
    • boolean flag = true;

      常用进制

      • 二进制 int a1 = 0b10; 输出2
      • 八进制 int a2 = 010; 输出8
      • 十进制 int a3 = 10; 输出10
      • 十六进制 int a4 = 0x10; 输出16

        类型转换

        java是强类型语言，不同类型变量计算时要进行类型转换。
    • 强制转换
      （类型） 变量名 高–>低
    • 自动转换
      低–>高
      注意：不要溢出变量类型的取值范围

JDK、JRE、JVM

JDK : Java Development Kit

（java开发工具包）

JRE : Java Runtime Environment

（java运行时环境）

JVM : JAVA Virtual Machine

（java虚拟机）

卸载jdk步骤

1.删除java安装目录
2.删除环境变量中JAVA_HOME
3.删除path下java目录
4.java-version

下载jdk及配置

1.官网下载1.8安装包
2.配置环境变量 系统变量JAVA_HOME、目录
3.配置path变量
4.在cmd输入 java -version查看是否下载成功

Markdown语法

文件后缀 xxx.md

1.标题：

#+标题内容 （加号表示空格）

一级标题

##+标题内容

二级标题

三级标题

。。等
#的个数1-6分别表示不同的标题等级

字体:

  • 斜体 *+内容+* （快捷键Ctrl+i）

test word

  • 加粗 *+内容+* （快捷键Ctrl+b）

test word

  • 斜体加粗 ***+内容+***

test word

  • 删除线 ~~+内容+~~
    test word
  • 引用 >+内容

    test word.

  • 分割线 *** 或 —(为虚线，hr+单括号为实线)
  • 图片 ! [图片无法显示时的文字] (图片路径或链接)

    * 超链接 [显示文字](链接)
    baidu
    * 表格
    1 2 3
    3 4 5
    5 6 7 
    * **代码** \```+java（语言名）+```
