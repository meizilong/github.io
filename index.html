*值传递与引用传递的区别

方法重载

  • 1.方法名必须相同
  • 2.参数的参数类型、参数个数或参数顺序（同方法内的不同参数类型）必须不同
  • 3.返回类型、修饰符可以相同、也可以不同
  • 4.仅有返回值不同不足以区分两个方法体

    可变参数（不定向参数）

  • JDK 1.5开始，java支持传递同类型的可变参数
  • 在方法声明中，在指定参数类型后加一个省略号“…”
  • 一个方法中只能制定一个可变参数，他必须是方法的最后一个参数。任何普通的参数必须在它之前声明。

    递归

  • 递归头：什么时候不调用自己
  • 递归体：什么时候调用自身

运算符

位运算符

位运算用二进制计算，效率高。

  • a = 0011 1100
  • b = 0000 1101
  • a&b = 0000 1100
  • a|b = 0011 1101
  • a^b = 0011 0001
  • ~a = 1100 0011

2*8 计算机如何运算效率最高？

  • << 左移 相当于*2（二进制运算）
  • >> 右移 相当于/2

    2*2*2*2 左移四位即可

    扩展赋值运算符

  • a+=b //a=a+b
  • a-=b //a=a-b

    条件运算符

  • x ? y : z //如果x为真，结果为y，否则为z。(重要)

    ==注意：多用（）减少bug #F44336==

变量

  • 数据类型有基本类型和引用类型
  • 变量名必须合法，不能用关键字，只能以_，$和字母开头

    常量

    作用域

  • 局部变量：必须声明和初始化值
  • 实例变量：从属于变量，如果不进行初始化，这个值默认为0或0.0

1.Boolean值默认为false也就是0

变量的命名规范

  • 类成员变量：首字母小写和驼峰原则（除了第一个单词外后面单词首字母大写 varName）

  • 常量：大写字母和下划线
  • 类名：首字母大写和驼峰原则
  • 方法名：首字母小写和驼峰原则

java数据类型

  • java为强类型语言，要求按严格要求使用变量，所有变量 都必须先定义，在使用。运行慢，但安全。
  • 弱类型语言则相反，没有严格的要求限制使用变量。运行快，但不太安全。

    八大基本数据类型

    整数
    • int a1 = 10;
    • byte a2 = 20;
    • short a3 = 30;
    • long a4 =40L;
      小数（浮点数）
    • float a5 = 1.25F;
    • double a6 = 2.354;
      浮点数只能表示有限的位数，位数大的二进制没有对应精确的值表示，会取最接近的数表示。
      所以最好要完全避免使用浮点数进行比较
      字符
    • char a7 = ‘a’ 单字符（字符串 String 属于类）
      布尔类型
    • boolean flag = true;

      常用进制

      • 二进制 int a1 = 0b10; 输出2
      • 八进制 int a2 = 010; 输出8
      • 十进制 int a3 = 10; 输出10
      • 十六进制 int a4 = 0x10; 输出16

        类型转换

        java是强类型语言，不同类型变量计算时要进行类型转换。
    • 强制转换
      （类型） 变量名 高–>低
    • 自动转换
      低–>高
      注意：不要溢出变量类型的取值范围
